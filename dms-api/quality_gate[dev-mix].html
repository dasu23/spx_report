主要 drivermgmt-domain/src/main/java/io/speedx/drivermgmt/domain/service/SourceRouteService.java:103-142
`getSourceRouteDetail` 在未做分页或上限控制的情况下，一次性加载指定 `sourceRouteId` 下全部 `Parcel`，随后继续级联查询全部 `RouteItem`、`Route`、`Fleet`、`Driver`、`Deliver` 并组装完整响应。对于包含大量包裹的 source route，这会把单次查询放大为高内存占用和多表批量读取，存在被大数据量请求触发的可用性风险。
