    MAJORL14API契约

    修复建议

    不要在已对外使用的 BO 上直接启用 `@Accessors(chain = true)`。保留原有标准 setter 签名；如确需链式调用，新增独立 builder/转换器，或显式保留兼容的 `void setXxx(...)` 方法。
