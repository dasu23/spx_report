    修复建议与测试场景
    修复建议

    将这些前置校验统一改回业务异常，例如 `if (req.getRouteId() == null && req.getDispatchId() == null) { throw new BizException("routeId or dispatchId can not be null"); }`，以及 `orElseThrow(() -> new BizException(LocalMsgUtil.getMsg("loading.route.doesNot-exist")))`。对路线状态校验也不要使用 `Assert`，改为显式 `if (...) throw new BizException(...)`。

    测试场景
