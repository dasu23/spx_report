    1. 数据范围过滤（最高优先级）
    - spx-operator：所有查询必须带 facilityCode in [用户 facilityList] 过滤
    - driver-leader：所有查询必须带 fleetId in [用户 fleetList] 过滤
    - spx-admin / admin-user：无需额外过滤

    2. 接口必须有权限注解
    - 使用 @RequirePermissions 或等效注解
    - 修改/删除接口必须校验资源归属（操作前确认资源属于当前用户的 facility 或 fleet）
