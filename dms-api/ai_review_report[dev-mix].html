major249-252边界条件登录校验直接调用 `driver.getPassword().equals(...)`。对历史司机数据或未设置过新字段 `driver.password` 的账号，这里会触发空指针，登录返回异常而不是明确的用户名/密码错误。先做空值保护，再比较密码，例如 `String hashed = ...; if (driver == null || StringUtils.isEmpty(driver.getPassword()) || !StringUtils.equals(driver.getPassword(), hashed)) { ... }`。同时为存量账号准备回填或兼容旧密码来源的迁移逻辑。构造一个 `driver.password = null`、但司机记录存在的账号，调用 `/login`，断言返回受控的认证失败响应而不是 500/NPE。
