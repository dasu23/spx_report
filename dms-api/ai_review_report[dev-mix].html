    将 SQL 中的表名改为实际物理表名，并与 `@Table` 显式保持一致，例如：`@Table(name = "parcel")` 配合 `@SQLDelete(sql = "update parcel set deleted = id where id = ?")`。这样可以避免命名策略或大小写差异带来的运行时失败。

    测试场景

    补充一个软删除集成测试：创建 `Parcel` 记录后调用 repository.delete(...)，断言 SQL 能成功执行且记录未物理删除、`deleted` 字段被更新。
