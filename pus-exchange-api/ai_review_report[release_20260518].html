    MAJORL24-27异常处理
    `limitFor` 直接对外部配置绑定的 `limit` 调用 `entrySet()`，未处理配置缺失或刷新期间为空的情况。若 `report.task.limit` 未配置、配置绑定失败，或 `@RefreshScope` 刷新瞬间出现空值，这里会触发 `NullPointerException`，导致报表任务请求异常失败，形成可被错误配置触发的可用性风险。
