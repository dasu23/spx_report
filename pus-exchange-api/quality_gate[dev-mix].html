`Runnable` 任务包装后把 `Exception` 和 `OutOfMemoryError` 仅记录日志而不再抛出，导致 `executor.submit()` 返回的 `Future` 会表现为正常完成。调用方即使执行 `Future.get()` 也无法感知任务失败，容易把实际执行失败误判为成功。
