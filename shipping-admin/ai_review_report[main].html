    修复建议

    在组装新值前兜底数组值，例如 `const currentValue = Array.isArray(value) ? value : []; const newValue = multiple ? [...currentValue, entryValue] : entryValue;`，或恢复 `value = []` 的默认值。

    测试场景

    以 `multiple={true}` 渲染组件且不传 `value`，上传一个文件，验证不会抛异常且 `onChange` 能收到包含新文件的数组。

    测试场景

    设置 `maxCount=3`，当前已有 2 张图片时一次粘贴 2 张，验证最多只接受 1 张且不会发起第 2 张上传请求。
