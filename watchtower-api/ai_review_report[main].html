    修复建议

    为该接口定义明确的响应 DTO，并让 Feign 直接反序列化为强类型对象，例如新增 `SlowQueryTriggerResponse`，包含 `fetchedCount`、`skippedCount`、`newIssuesCount`、`reopenedCount` 等字段，然后将方法签名改为 `SlowQueryTriggerResponse trigger(@PathVariable("id") Long id);`。

    测试场景
