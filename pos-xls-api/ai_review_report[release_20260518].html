    MAJORL59-75并发安全
    `run()` 先阻塞等待 `producerFuture.get()`，再等待 `consumerFuture.get()`。如果消费者先失败，而生产者此时因有界队列写满而阻塞在 `queue.put()`，当前线程会一直卡在 `producerFuture.get()`，整个流水线无法返回失败结果。
